# Raindour - Smart Audio Player
**By Aotsuki Lab**

An ultra-lightweight audio/video player engineered for high performance.

### 🚀 Smart Features
- **Adaptive FPS:** Detects gaming/coding focus to drop resource usage.
- **Multi-Era Support:** Optimized for Windows 7 through Windows 11.
- **Hybrid UI:** Picture-in-picture and ambient blur modes.
